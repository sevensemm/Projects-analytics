# Аналитика продуктивности сотрудников

Веб-сервис для строительной компании. Позволяет управлять проектами, задачами и отслеживать эффективность работы команды.

---

## Основной функционал

**Для всех сотрудников**
- Просмотр своих проектов и задач
- Изменение статуса задачи (выполнена / не выполнена)
- Добавление комментариев к задачам
- Фиксация фактической даты выполнения

**Для администратора**
- Создание и удаление проектов и этапов
- Назначение задач любым сотрудникам
- Управление пользователями (добавление / удаление)
- Аналитическая панель с графиками и метриками

**Аналитика**
- Круговые диаграммы распределения выручки по проектам и этапам
- Сравнение плановых и фактических трудозатрат
- Пузырьковые графики эффективности сотрудников
- Расчёт отклонений от плана и ROI
